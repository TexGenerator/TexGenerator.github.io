Abstract

In this paper, we present TextureGenerator, an innovative framework designed to generate high-quality, semantically consistent 3D texture maps using reference images. While text-based 3D texture generation has seen significant advancements, challenges persist when using images as supervisory signals, often resulting in subpar outcomes. Existing methods primarily rely on pre-trained Text-to-Image (T2I) diffusion models in combination with the IP_Adapter style transfer model for texture generation. However, these approaches tend to capture only a fraction of the image's stylistic features, leading to difficulties in accurately reproducing texture details from the reference image, and thereby yielding lower-quality results. To address these challenges, the proposed TextureGenerator framework introduces a novel approach: it begins by employing a rapid personalization diffusion model based on the reference image, modifies the conditional image in Canny ControlNet to align its line semantic distribution with the reference image, and finally integrates a multi-view diffusion model to generate precise 3D textures that preserve both style and semantic consistency. Compared to current state-of-the-art methods, TextureGenerator markedly improves the similarity and quality of textures generated from reference images, offering robust support for the creation of realistic, high-quality 3D assets..
